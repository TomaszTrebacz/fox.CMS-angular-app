## Description

Angular-app enabling user interacting with backend api.

## Installation

1. Create .env file following the example (example.env) in the repository.

```ts
AES_KEY=
BACKEND_URL=

AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_REGION=

```

2. Install dependencies

```
npm install
```

3. Run the app:

```ts
ng install
```

4. Navigate to `http://localhost:4200/`
